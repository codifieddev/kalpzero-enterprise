# 09 AI Foundation Spec

## Wave 1 AI Scope

- Tenant-scoped provider configuration
- Model policy and limits
- Knowledge ingestion pipeline
- Usage metering
- Auditability
- Prompt and tool boundaries

## AI Rules

- No public unauthenticated chat endpoints
- No shared provider key usage without tenant policy
- Human approval for sensitive workflows
- PII redaction and document classification before retrieval

## Initial Use Cases

- Operator assistant for platform navigation and knowledge retrieval
- Draft generation for travel itineraries and publishing copy
- Import diagnostics and mapping support

Advanced agentic automation is deferred until the canonical data and control
boundaries are stable.
